Recent Projects

Cache Controller

Digital Systems Engineering
Designed and implemented a direct mapped memory cache controller on the Xilinx Spartan 3E FPGA. Cache controller designed provided experience in the design of high performance cache controllers, clock synchronization techniques and design simulation/verification methods.

VGA Interface Controller

Digital Systems Engineering
Designed and implemented a VGA controller interface to display a Pong video game on an LCD. The design provided experience in the design and implementation of high performance signal generators, clock synchronization, external I/O and on-chip interfacing using custom logic circuits on the Xilinx Spartan 3E FPGA.

Multimedia Center Project

Embedded Systems Engineering
Designed and implemented a multimedia environment on an ARM Cortex M3 with interfacing to external I/O and on-chip hardware. The multimedia center provided a full user experience with an integrated menu for selecting a specific application to run within the project. The applications available were a gallery/slideshow, audio streaming from the host computer and a video game (Pong). Users were able to manipulate and traverse through the menu using onboard input controls such as buttons and twist knobs, an small LCD screen was used to display the various applications to the user. Memory/timing optimization techniques (bit-banding, conditional branching) were used along with other ARM specific optimizations to improve user experience, especially under the graphically intensive video game.

Peer to Peer Client-Server Project

Computer Networks
Designed and implemented a peer to peer system with an index server for file tracking and client-servers for file transfer and communication using the C programming language. A client side program was provided with a command-line based user interface and multiplexing techniques (POSIX file descriptors, streams) were used to switch between file transfer (peer to peer) and local user input. Client side program required the use of both UDP and TCP protocols; speed and efficiency was obtained using UDP when communicating with the index server while TCP provided a reliable method of file transfer between peers.
